# Sustainability FPS (Three.js)

First-person shooter prototype themed around sustainability, set on a failing oil rig overrun by Oil zombies.

## Project Docs

- Requirements: `docs/PROJECT_REQUIREMENTS.md`
- Architecture and file layout: `docs/PROJECT_STRUCTURE.md`

## Current Status

- Detailed MVP and system requirements are defined.
- Modular source tree is scaffolded for rapid feature development.

## Run Locally

1. Install dependencies:
	- `npm install`
2. Start the dev server:
	- `npm run dev -- --host 0.0.0.0 --port 4173 --strictPort`
3. Open localhost quickly from this environment:
	- `npm run open`
4. Direct URL:
	- `http://localhost:4173/`

## If localhost does not open

- In VS Code, open the **Ports** panel.
- Find port `4173` and click **Open in Browser**.
- Use that forwarded URL instead of local machine `localhost`.
