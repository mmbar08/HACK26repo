# Sustainability FPS (Three.js)

First-person shooter prototype themed around sustainability, set on a failing oil rig overrun by Oil zombies.

## Project Docs

- Requirements: `docs/PROJECT_REQUIREMENTS.md`
- Architecture and file layout: `docs/PROJECT_STRUCTURE.md`

## Current Status

- Detailed MVP and system requirements are defined.
- Modular source tree is scaffolded for rapid feature development.

## Run Locally

- Install dependencies: npm install
- Start dev server: npm run dev -- --host 0.0.0.0 --port 4173 --strictPort
- Open game page quickly: npm run open
